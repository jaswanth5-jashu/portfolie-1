Portfolio Website

About the Project

This Personal Portfolio Website is designed to showcase my skills, projects, and professional identity in a clean and modern way. The website is fully responsive and built using HTML, CSS, and JavaScript to ensure smooth performance across all devices. It includes a visually appealing hero section, project gallery, detailed project descriptions, and a contact page for recruiters and clients to reach out easily.

The portfolio reflects my design approach—simple, attractive, and user-friendly—while integrating animations, transitions, and interactive UI elements. With features like dark/light mode, responsive layout, and external project links, this website serves as a digital representation of my work, personality, and technical abilities.
