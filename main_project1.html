About the Project

The Real-Time Traffic Surveillance & Detection System uses Deep Learning and Computer Vision to monitor vehicle activity and identify abnormal driving behavior in real time. The system analyzes traffic footage, detects vehicles, and classifies actions such as overspeeding, sudden braking, wrong-way movement, and lane violations using models like CNN, SVM, NaÃ¯ve Bayes, and Random Forest.
